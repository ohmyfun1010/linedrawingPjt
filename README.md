1명

<br>

### 구현 기능

     - 로그인기능
     - CRUD
     - 카페24 tomcat 웹호스팅
     
     

<br>

### 개발 도구

     - Jdk : 8 버전
     - Spring framework : spring 5
     - 웹서버 : Apache
     - WAS : Tomcat 9.0
     - Maven : 2.9
     - DBMS : mariadb 10.1.13

## 프로젝트 소개

### 라인드로잉 소개 및 리뷰 게시판

- 라인드로잉 주문전 리뷰 및 예제 상품들을 볼 수 있습니다.
- 공지게시판의 공지글은 오직 관리자만 작성,수정,삭제가 가능합니다.(관리자 id:root pw:root)
- 로그인 후 리뷰 게시판의 글을 작성 할 수 있으며 검색 기능을 통해 작성자의 글을 한번에 찾을 수 있습니다.
